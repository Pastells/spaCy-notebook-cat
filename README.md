# Un quadern sobre el curs de PLN d'Spacy

## Sobre aquest quadern

Aquest quadern es basa en https://github.com/Cristianasp/spacy-notebook, alhora basat en el curs en línia gratuït d'Ines Montani, disponible a:

https://course.spacy.io/

https://github.com/ines/spacy-course

## Com instal·lar spacy:

Les instruccions es troben a l'enllaç següent:

https://spacy.io/usage

Per instal·lar spacy, utilitzeu:

<pre><code>pip install spacy</code></pre>

Aquest quadern s'ha provat per a la versió 3.8.11

Per comprovar la versió de la biblioteca, utilitzeu les ordres següents:

<pre><code>import spacy
print(spacy.__version__)
</code></pre>

## Com descarregar els models:

https://github.com/explosion/spacy-models/releases/

Per descarregar un model:

<pre><code>python -m spacy download en_core_web_sm
python -m spacy download ca_core_news_sm
</code></pre>
